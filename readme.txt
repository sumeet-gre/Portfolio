Organization

-linked_list.h - Definition of the linked list functions
-linked_list.c - Code for linked list functions
-main.c - Tester for linked list functions
-bubblesort.o - Sorting file to test functions
-Makefile - Generate code

Objective
=========
A linked list stored and used to manipulate records of employees at Carleton University.

Purpose
=======
-To demonstrate insertion to a linked list
-To search a linked list
-To delete from a linked list
-To demonstrate the implementation and manipulation
 of a linked list
-To understand memory allocation

Instructions
============
Compile the program with the following:
"gcc -g main.c linked_list.c bubble_sort.o"
With main.c being your test program
